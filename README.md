# My Portfolio

This project is a personal portfolio website designed to showcase my work as a creative developer. The website features a modern aesthetic with a video background that enhances the visual appeal while maintaining readability.

## Project Structure

```
my-portfolio
├── index.html         # The main HTML file containing the structure of the portfolio
├── style.css          # CSS file for styling the portfolio
├── scripts
│   └── main.js        # JavaScript file for interactive functionality
├── assets
│   └── videos
│       ├── background-loop.mp4  # Video file for the background
│       └── background-loop.webm  # WebM format video for fallback
└── README.md          # Documentation for the project
```

## Features

- **Responsive Design**: The portfolio is designed to be responsive and looks great on all devices.
- **Video Background**: A looping video background that transitions from a white screen to a dark screen, enhancing the aesthetic appeal.
- **Interactive Elements**: JavaScript functionality to enhance user interaction.

## Setup Instructions

1. Clone the repository to your local machine:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd my-portfolio
   ```

3. Open `index.html` in your web browser to view the portfolio.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Video Formats (MP4, WebM)
